# Universal Doc Converter Pro 📄➡️📝

Современное десктопное приложение на Python для конвертации документов в различных направлениях. Поддерживает работу с PDF, Word, Markdown и текстовыми файлами.

## Основные возможности

- **Универсальная конвертация**: 
  - Из PDF, DOCX, XLSX, PPTX в Markdown.
  - Из Markdown и TXT в DOCX (Word), PDF, TXT.
- **Drag-and-Drop**: Просто перетащите файл в окно программы для начала работы.
- **Встроенный редактор**: Вы можете редактировать текст после конвертации или написать его с нуля прямо в приложении перед сохранением.
- **Современный интерфейс**: Поддержка темной и светлой тем, интуитивно понятное управление.
- **Предпросмотр**: Мгновенное отображение содержимого документа.

## Стек технологий

- **Python 3.11+**
- **CustomTkinter**: Современный графический интерфейс.
- **MarkItDown (Microsoft)**: Мощный движок для парсинга документов.
- **Pandoc + Pypandoc**: Для создания безупречных .docx файлов.
- **TkinterDnD2**: Поддержка перетаскивания файлов.
- **FPDF2**: Генерация PDF документов.
